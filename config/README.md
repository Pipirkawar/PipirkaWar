# config/

Настройки и конфигурации проекта.

- `balance.yaml` (TBD в Спринте 0.2) — все балансовые числа: цены толщины, кулдауны, множители каравана, бонус «Главы клана дня» и т. д.
- `locales/` — fluent-файлы локализации (`ru.ftl`, `en.ftl`, …).

Секреты — **только в `.env`** (см. `.env.example` в корне). В этой папке секретов нет.
