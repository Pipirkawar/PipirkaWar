- **D.10.b-3 (этот коммит):** Реальные TEP-67 wallet-v3R2 external-message + TEP-74 jetton-transfer-body в `TonRpcAdapter`:
  - **Constructor:** `TonRpcAdapter(client, settings, jetton_provider, signer: ITonMessageSigner, clock: Callable[[], float] | None = None)`. `signer` инжектится через DI (в composition root в D.10.c). `clock` для детерминизма в тестах; default `time.time`.
  - **`_fetch_seqno(payout_wallet)`:** вызывает `client.run_get_method(address=payout_wallet, method="seqno", stack=())`; бросает `TonRpcCallError` при `exit_code != 0` или пустом stack-e.
  - **`_derive_query_id` (переписан):** `hashlib.blake2b(f"{currency.value}|{recipient}|{amount_native}".encode(), digest_size=8)` → 64-bit BE int. Стабильнее чем Python `hash()` (cross-process). Не cryptographic-strong, caller берёт dedup-гарантию через `IIdempotencyKey`.
  - **`_build_ton_internal_message(amount, recipient)`:** TEP-67 int_msg_info$0, bounce=0, src=addr_none, dest=`MsgAddressInt(recipient)`, coins=amount, init=empty, body=empty (0-bit). Cell без refs.
  - **`_build_jetton_internal_message(jetton_wallet, internal_ton_amount, jetton_body_cell)`:** TEP-67 int_msg_info$0, bounce=1, dest=`MsgAddressInt(jetton_wallet)`, coins=internal_ton (100M nano = 0.1 TON), body как ref на `jetton_body_cell`.
  - **`_build_jetton_transfer_body(op_code, query_id, amount, dest, response_dest, forward_ton)`:** TEP-74 op-code(32) + query_id(64) + amount (VarUInt 16 = `store_coins`) + dest (MsgAddressInt) + response_dest (MsgAddressInt) + custom_payload-flag(1 bit 0) + forward_ton (`store_coins`) + forward_payload-flag(1 bit 0). Op-code жёстко зафиксирован `0x0F8A7EA5`; bad-op → `ValueError`.
  - **`_build_signed_external_message_boc(payout_wallet, seqno, internal_message)`:** wallet-v3R2 external-message:
    1. `unsigned_body` = subwallet_id(32) + valid_until(32, = clock() + 60) + seqno(32) + send_mode(8, = 3) + ref(internal_message).
    2. `signature = signer.sign(message=unsigned_body.repr_hash())` → 64 bytes.
    3. `signed_body` = signature(512 bits) плюс всё из unsigned_body. **Сигнатура — в inline-bits, не в ref-е** (совпадает с wallet-v3R2 reference).
    4. `external_message` = ext_in_msg_info$10 + addr_none(src) + MsgAddressInt(payout_wallet) + import_fee=0 + init-flag(0) + body-flag(1 ref-flag) + ref(signed_body).
    5. `base64.b64encode(serialize_boc(external_message)).decode("ascii")`.
  - **`payout()` rewritten:**
    - TON: `internal = _build_ton_internal_message(amount, recipient); seqno = _fetch_seqno(payout_wallet); boc = _build_signed_external_message_boc(...); client.send_boc(boc)`.
    - USDT: jetton-wallet через `jetton_provider.resolve_jetton_wallet(recipient)`, затем jetton_body → internal → seqno → signed BoC.
