## 🛑 Известные блокеры / открытые вопросы PR-а

- **Flaky `test_invalid_payload_logs_machine_readable_reason` в `tests/unit/bot/handlers/test_roulette_paid.py` (closes-on F.1).** В `make ci` ~1% флейков, в изоляции (`pytest <file>::<test>`) — стабильно зелён. PR #133 CI красный по этой же причине, мерж был выполнен принудительно. F.1 закрывает.
- **`SandboxTonConnectVerifier` (4.1-D backlog → главный таргет 4.1-F).** В composition root живёт stub-верификатор: `is_sandbox=True` → accept-any-non-empty-proof; `is_sandbox=False` → fail-closed + WARNING-log. Это **блокер для open-launch-а в prod**.
